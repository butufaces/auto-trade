## 🎨 Color & Style Guide

### Button Styles (Telegram Inline Keyboards)
- **Primary Actions**: ✅ Green (Approve, Confirm, Send)
- **Destructive**: ❌ Red (Reject, Delete, Cancel)
- **Navigation**: 🔵 Blue (Next, Back, Details)
- **Neutral**: ⚪ Gray (Skip, Close, Info)

### Status Indicators
- ✅ **ACTIVE** - Green checkmark
- ⏳ **PENDING** - Hourglass
- ❌ **REJECTED** - Red X
- 🟢 **COMPLETED** - Green circle
- ⏸️ **SUSPENDED** - Pause icon
- 🔴 **DELETED** - Red circle

### Icons Used
- 💰 Money/Packages
- 📊 Portfolio/Dashboard
- ⏳ Pending/Time
- 👥 Users
- 📣 Announcements
- 📋 Logs/History
- ⚙️ Settings
- ❓ Help
- ⭐ Reviews
- 🏠 Home

## 📱 User Journey Example

User invests $250 in Starter package:

```
1. /start
   ↓
2. Main Menu (select 💼 Invest)
   ↓
3. Select Package (Starter)
   ↓
4. Package Details (confirmation)
   ↓
5. Select Amount ($250)
   ↓
6. Confirm Investment (Review)
   ↓
7. Success + Status: ⏳ PENDING
   ↓
8. [Admin approves within 24 hours]
   ↓
9. Status: ✅ ACTIVE (User notified)
   ↓
10. [After 30 days]
    ↓
11. Status: 🟢 COMPLETED + Profit
    ↓
12. User views in 📊 Portfolio
    ↓
    Earned: $25 profit ✓
```

## 🔐 Admin Authority Levels

```
Regular User:
├─ 💼 Invest (Create investments)
├─ 📊 View Portfolio
├─ ⚙️ Settings
└─ ❓ Help

Admin (/admin):
├─ 💰 Manage Investments (Approve/Reject)
├─ 👥 Manage Users (Suspend/Delete)
├─ 📣 Send Announcements
├─ 📋 View Activity Logs
└─ ⚙️ System Settings

Super Admin (Optional):
├─ [All Admin features]
├─ 🔧 System Configuration
├─ 📊 Export Reports
└─ 🗑️ Archive Data
```
